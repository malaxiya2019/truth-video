# Truth Router v6

## AI Hallucination

Large Language Models sometimes generate confident but incorrect facts. This is called hallucination.

## Knowledge Graph Solution

Truth Router validates every claim against a structured Knowledge Graph using a Verification Pipeline.

## Key Features

Real-time verification, confidence scoring, and automatic error correction. Supports RAG and API integration.

## Use Cases

Technical documentation, product tutorials, educational content, and AI-generated explainer videos.
